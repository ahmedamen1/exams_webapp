# منصة امتحانات أ/ أحمد سرور

تم بناء نسختين داخل نفس المشروع:

- `index.html` + `static-app.js`: نسخة تعمل فورا بدون أي تثبيت، ببيانات محلية في `localStorage`.
- `src/app`: هيكل Next.js مطابق لوثيقة المعمارية، جاهز للربط مع Firebase بعد تثبيت الحزم وإضافة مفاتيح `.env.local`.

## التشغيل الفوري بدون تثبيت

```powershell
node server.js
```

ثم افتح:

```text
http://localhost:3000
```

روابط الاستخدام:

```text
رابط الطلاب: http://localhost:3000/index.html
رابط لوحة التحكم: http://localhost:3000/dashboard.html
```

كود دخول لوحة التحكم في نسخة الديمو:

```text
1234
```

كود الامتحان التجريبي:

```text
MATH3A
```

## تشغيل نسخة Next.js

```powershell
npm.cmd install
npm.cmd run dev
```

ثم افتح:

```text
http://localhost:3000
```

> ملاحظة: في هذه البيئة، `npm install` انتهى بمهلة زمنية قبل تنزيل الحزم، لذلك تم توفير النسخة الثابتة لتجربة المنصة مباشرة.

## ما تم تنفيذه

- واجهة عربية RTL كاملة بألوان الوثيقة وشعار `logo.png`.
- دخول الطالب بكود الامتحان.
- إدخال بيانات الطالب ورقم التليفون وبدء الامتحان.
- مؤقت امتحان مع تسليم تلقائي عند انتهاء الوقت.
- أسئلة اختيار من متعدد ومقالي.
- الطالب يمكنه في السؤال المقالي كتابة الإجابة وإرفاق صورة من الكاميرا أو من الجهاز.
- إضافة سؤال كنص أو كصورة مرفوعة من لوحة التحكم.
- بنك أسئلة مستقل يمكن إعادة استخدامه في أي امتحان.
- سحب سؤال من بنك الأسئلة وإفلاته داخل الامتحان الحالي.
- إضافة أسئلة كثيرة مرة واحدة من مربع الإدخال الجماعي.
- اختيارات MCQ يمكن أن تكون نصا أو صورة لكل اختيار.
- الوضع العشوائي يدعم أسئلة متكافئة: ضع أكثر من نسخة لنفس رقم/مجموعة السؤال، والنظام يختار نسخة واحدة عشوائيا لكل طالب مع الحفاظ على نفس مستوى الصعوبة.
- توليد نموذج إجابة من الامتحان لمشاركته مع الطلاب أو طباعته PDF.
- السؤال المقالي يدعم نموذج إجابة نصي أو صورة إجابة نموذجية مرفوعة من المدرس.
- ترتيب عشوائي ثابت حسب اسم الطالب.
- تصحيح MCQ تلقائي.
- نتيجة الطالب مع حالة المقالي.
- لوحة تحكم للمدرس بتصميم ورابط منفصل عن واجهة الطالب.
- إنشاء امتحانات وإضافة أسئلة.
- جدول نتائج متقدم مع رقم التليفون، عرض إجابات كل طالب، وتصحيح مقالي.
- تحليلات ذكية للنتائج: متوسطات، توزيع درجات، نسبة نجاح، أكثر الأسئلة خطأ، وتنبيهات مراجعة.
- تصدير CSV للنتائج وتصدير منفصل لمراجعة إجابات الطلاب.
- قواعد Firestore و Storage في ملفات منفصلة.
